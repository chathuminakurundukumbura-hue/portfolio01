This is a mobile-first, animated Next.js portfolio scaffolded for Chathumina Bandara.

Quick start

1. Install dependencies:

```bash
npm install
```

2. Run dev server:

```bash
npm run dev
```

Open http://localhost:3000

Notes
- Replace `/public/profile-placeholder.svg` with your real photo (keep filename or update `Hero.js`).
- The project uses `framer-motion` for animations.
- To deploy: push to GitHub and use Vercel (zero-config) or Netlify with Next.js build settings.
